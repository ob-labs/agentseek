# AgentSeek documentation

AgentSeek helps you run an agent locally, turn it into an editable project,
and add runtime capabilities when the project needs them.

## First path

1. [Run the CLI quick demo](tutorials/01-quick-demo-cli.md).
2. [Create your first harness app](tutorials/02-first-harness-app.md).
3. [Add a skill and MCP](tutorials/03-add-a-skill-and-mcp.md).

This path keeps the first run small. It introduces configuration, projects,
skills, and MCP only when they are needed.
