# AgentSeek 文档

AgentSeek 帮你先在本地跑通 agent，再把它变成可编辑项目；当项目需要时，
再加入 runtime 能力。

## 第一条路径

1. [运行 CLI 快速演示](tutorials/01-quick-demo-cli.zh.md)。
2. [创建第一个 harness 应用](tutorials/02-first-harness-app.zh.md)。
3. [添加 skill 和 MCP](tutorials/03-add-a-skill-and-mcp.zh.md)。

这条路径把第一次运行保持得足够小。配置、项目、skill 和 MCP 都在需要时再出现。
