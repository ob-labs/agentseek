## 这些页面覆盖什么

- `agentseek` CLI
- `.agentseek` 下的项目默认布局
- `AGENTSEEK_*` 别名（兼容 Bub）
- `src/skills` 下的内置技能

`contrib/` 下的可选集成各自维护独立 README。若你想在一个地方查看插件、技能和相关链接，请使用 [目录](../hub.md)。
