# Frontend — My Bub Agent

CopilotKit + Vite frontend. Talks to the agentseek gateway over AG-UI
through a small Copilot Runtime (Express).

## Run

```bash
npm install
npm run dev
```

| Service | Port |
| --- | --- |
| Vite dev server | 5173 |
| Copilot Runtime | 4000 |
| Agentseek gateway | 8088 |
